# **App Name**: ExamFocus

## Core Features:

- Exam Registration: Easily add new exams, specifying subject, date, and optional notes, which will be stored for later access.
- Upcoming Exam Countdown: View prominent, visually engaging countdown timers for each registered upcoming exam.
- Organized Exam List: Access a clear, sorted list of all scheduled exams, allowing for quick overview.
- Exam Completion & Scoring: Mark exams as completed and record scores, contributing to a historical performance log.
- Performance Review Dashboard: Review a dashboard showing past exam results, averages, and basic performance insights.
- AI Study Plan Tool: Utilize an AI-powered tool to generate personalized study plan recommendations based on upcoming exam details and historical performance.

## Style Guidelines:

- Primary color: A calm, deep blue (#3E7AF7), evoking focus and clarity.
- Background color: A very subtle, cool white with a hint of blue (#F8F9FB), maintaining a clean and serene aesthetic.
- Accent color: A vibrant yet serene cyan (#179FD6), used for interactive elements and highlights to draw attention without being overwhelming.
- Body and headline font: 'Inter' (sans-serif) for its modern, clean, and highly readable characteristics, suitable for conveying information clearly to students.
- Employ a consistent set of line-based, minimalistic icons that are intuitively understandable for actions like 'add exam', 'mark complete', 'view history', and 'settings', aligning with the clean aesthetic.
- Adopt a clean, spacious layout with ample whitespace, ensuring information is digestible and easy to scan, especially for lists of exams and study plans. Prioritize responsive design for optimal viewing across devices.
- Incorporate subtle and smooth transitions for state changes, such as marking an exam complete, updating a countdown timer, or navigating between sections, enhancing the user experience without distraction.
